# Webinár 1 – Úvod (2025-09-01)

## Obsah
- Používanie terminálu vo Windowse
- Inštalácia Pythonu
- Používanie AI (ChatGPT, Claude AI, Gemini, Perplexity…)
- Spúšťanie Python interpreteru

## Poznámky
- Terminál vo Windowse:
  - spustenie cez **cmd** alebo **PowerShell**
  - základné príkazy: `cd`, `dir`, `cls`
- Inštalácia Pythonu:
  - inštalátor z [python.org](https://www.python.org/)
  - nezabudnúť zakliknúť **Add Python to PATH**
- AI nástroje:
  - využitie na generovanie kódu, vysvetlenie chýb, doplnenie poznámok
- Python interpreter:
  - spustenie: `python`
  - ukončenie: `CTRL+D` (Linux/macOS) alebo `CTRL+Z` + Enter (Windows)

## Úlohy
1. Spustiť terminál a vyskúšať základné príkazy.
2. Nainštalovať Python a overiť verziu (`python --version`).
3. Spustiť a ukončiť Python interpreter.
