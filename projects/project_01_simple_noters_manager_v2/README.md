# Simple Notes Manager v2

Terminal application written in Python for managing notes.

## About This Version

This is Version 2 of the Simple Notes Manager project.

I originally completed the first version as part of my Python course. The goal of this second version is not only to rebuild the project, but also to review and reinforce Python fundamentals such as functions, file handling, validation, refactoring, and project structure.

The original version will remain unchanged and serves as a reference point for comparison and measuring my progress.

This project is intended to evolve over time. Whenever I learn a new Python concept or think of an improvement, I may return to this project to refactor the code, add new features, and apply newly acquired knowledge.

For me, this project is both a practical application and a long-term learning playground.


## Features

* Add Note
* List Notes
* Delete Note
* Automatic ID generation
* Validation of user input
* Pause and clear screen
* Emoji terminal interface
