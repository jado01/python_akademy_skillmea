# Mini Calculator

Simple console calculator written in Python.

This project started as a very small menu-based calculator while revising Python basics.
Over time, I gradually improved and refactored the code to learn more advanced concepts step by step.

The project is mainly used for learning and practicing:

* functions
* loops
* conditions
* dictionaries
* file handling
* exception handling
* refactoring
* clean code
* helper functions
* modular project structure

## Features

* Addition
* Subtraction
* Multiplication
* Division
* History system
* Save history to TXT file
* Load saved history
* Clear history
* Timestamped calculations
* Error handling

## Run

```bash
python main.py
```
