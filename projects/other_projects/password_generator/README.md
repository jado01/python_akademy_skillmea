# Password Generator

## Project Goal

This project generates random passwords based on user preferences.

The user can choose:

* Password length
* Lowercase letters (a-z)
* Uppercase letters (A-Z)
* Numbers (0-9)
* Special characters (!@#$%^&*)

The program generates a random password according to the selected options.

## Learning Goals

This project is intended to practice:

* Functions
* Input validation
* Loops
* Conditions
* Strings
* Lists
* Python modules
* Random module

## Future Improvements

* Copy password to clipboard
